Lesson 0003

凸函数：把“形状”变成可证明的优化结构

对应 Boyd & Vandenberghe, Convex Optimization, Chapter 3: Convex functions。

本章的科研用法

第二章问“可行域是不是凸的”，第三章问“目标函数和约束函数是不是凸的”。从建模角度看，凸函数是把局部斜率、曲率、上图、保凸操作连接到全局最优性的语言。

一个定义

凸函数：函数 f 是凸的，如果 dom f 是凸集，并且对所有 x, y 属于 dom f 与 0 <= θ <= 1，有 f(θx + (1 - θ)y) <= θf(x) + (1 - θ)f(y)。

几何图像：函数图像上任意两点的弦，在函数图像上方。直觉是“插值后的函数值不超过函数值的插值”。

四条证明路线

路线 什么时候用 你要证明什么
定义 范数、max、简单表达式 直接证明 Jensen 型不等式。
限制到直线 矩阵函数、多变量函数难直接看 任意直线上的一元函数都是凸的。
微分判据 函数可微或二阶可微 一阶支撑超平面不等式，或 Hessian 半正定。
保凸操作 表达式由已知凸函数搭出来 非负加权和、仿射复合、逐点上确界、perspective 等保凸。

两个判据最常用

一阶判据

可微函数 f 凸，当且仅当 dom f 凸且 f(y) >= f(x) + ∇f(x)^T(y - x)。它说每个切平面都是全局下界。

二阶判据

二阶可微函数 f 凸，当且仅当 dom f 凸且 ∇²f(x) ⪰ 0。对二次函数，这直接变成矩阵 P ⪰ 0。

本章别漏的两个路标

上图 epigraph

f 凸等价于 epi f = {(x, t) | f(x) <= t} 是凸集。这个等价会把“函数凸性”转成“集合凸性”。

共轭 conjugate

f*(y) = sup_x (y^T x - f(x))。先把它当成“用线性函数从下方探测 f 的工具”，第五章对偶性会真正用上。

快速检索练习

要证明 f(x) = max{x1, ..., xn} 是凸函数，最自然的路线是哪一条？
