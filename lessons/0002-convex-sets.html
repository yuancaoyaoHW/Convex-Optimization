Lesson 0002

凸集：先判断可行域有没有全局结构

对应 Boyd & Vandenberghe, Convex Optimization, Chapter 2: Convex sets。

本章的科研用法

Chapter 2 不是在收集几何名词，而是在训练一个判断：你的可行域是否允许“在两个可行解之间插值仍然可行”。一旦这个判断成立，后面的凸函数、对偶性、最优性条件才有地方落脚。

一个检验

线段检验：集合 C 是凸的，当且仅当任取 x, y 属于 C，任取 theta 在 [0, 1]，都有 theta x + (1 - theta) y 属于 C。

这句话的含义很直接：如果两个方案都可行，那么它们的任何加权平均方案也可行。科研建模里，这常常对应“混合策略”“资源分配比例”“概率分布”“松弛变量”等结构。

三种集合别混

对象 封闭的组合 直觉
仿射集 theta x + (1 - theta) y, theta 为任意实数 整条直线都留在集合里，例如线性方程 Ax = b 的解集。
凸集 theta x + (1 - theta) y, 0 <= theta <= 1 只要求两点之间的线段留在集合里。
凸锥 theta1 x + theta2 y, theta1, theta2 >= 0 允许非负缩放和相加，例如正半定锥。

本章最有用的工具箱

常见例子

超平面、半空间、范数球、多面体、正半定锥。这些是以后建模时最常见的可行域积木。

保凸操作

交集、仿射映射、透视函数、线性分式映射等能保凸。看到复杂约束时，优先把它拆成这些操作。

分离思想

两个不相交凸集可以被超平面分开。后面的对偶证书、KKT 条件、支持超平面都会反复用到这个图像。

读这一章时怎么用力

不要平均用力背每个定义。你的第一遍目标是：看到一个集合，能用“线段检验”或“保凸操作”说明它为什么凸；如果不能说明，要知道卡在哪里。

小练习
